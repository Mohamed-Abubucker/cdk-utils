ConstructBuilder

ConstructBuilder

Class that can be used to traverse a file system, load constructs from individual files, initialize and then configure them. Constructs can be defined in individual files, but must be wrapped with a ConstructFactory instance. Any file that does not export a ConstructFactory instance will be ignored.
